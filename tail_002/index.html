Box Model Exercises with Tailwind CSS

Exploring borders, padding, margins, and box-sizing

Task 1: Different Border Widths and Colors

This box has a 4px red top border, 8px blue bottom border, 2px green left border, and 6px yellow right border.

Tailwind’s border-[side]-[width] and border-[side]-[color] classes allow precise control over each border.

Task 2: Padding Changes

No padding: Content touches the edges.
With 1.5rem (24px) padding: Notice the space inside the box.

Tailwind’s p-[size] class adds padding uniformly. Compare the two boxes to see the effect.

Task 3: Margin Spacing

Box 1: Default spacing.
Box 2: 2rem (32px) top margin for extra spacing.
Box 3: 3rem (48px) top margin for even more spacing.

Tailwind’s mt-[size] and space-y-[size] classes control margins and spacing between elements.

Task 4: Box-Sizing: Border-Box

Without box-border: Width is 256px + padding + border (total > 256px).
With box-border: Width stays 256px, including padding and border.

Tailwind’s box-border applies box-sizing: border-box, ensuring padding and borders don’t increase the element’s size beyond its set width.
